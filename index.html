Methodology & Sources

This application uses the Full 50-State Dataset (v2.5). Metrics are synthesized from the following primary sources:

  • Fiscal: Tax Foundation (2025 Index)
  • Schooling: HSLDA Regulatory Tiers
  • Regulatory/Land Use: Cato Institute: Freedom in the 50 States
  • Zoning Friction: Mercatus Land Use Index

Domain friction is computed by mapping these indices to action-based components: permission count, wait times, and penalties. View Raw Dataset (JSON)

The Engineering Model
1. Action Friction = (0.4 × Permissions) + (0.4 × Time) + (0.2 × Penalty)
2. Domain Friction = Average(Action Frictions)
3. Freedom = 1 - [(Fiscal Wt × Tax Burden) + (Perm Wt × Reg Friction)]
